My Projects

Multi-threaded C++ Physics Simulator

Developed a high-performance, multi-threaded C++ physics simulator for granular gas dynamics. The project focuses on optimized code for complex physical simulations.

This work will be presented at IAC 2025, showcasing the application of advanced computing techniques to aerospace physics problems.

Rocket Control System Firmware

Developed embedded firmware for a real-time multithread rocket control system, integrating sensor data processing and control using FreeRTOS and ESP-IDF.

This project involved implementing robust control algorithms and optimizing for the resource constraints of embedded systems.

Aerodynamic Simulation Analysis

Performed, analyzed, and helped develop interpolation algorithms for aerodynamic simulations of rockets using Pandas, Matplotlib, TensorFlow and Ansys.

Co-authored research on compression lift analysis for the SpaceCab project, applying computational methods to understand complex aerodynamic phenomena.

Drag Coefficient Study

Conducted a study of drag coefficient as a function of Mach number for a rocket model. Results were compared with experimental data from Modern Exterior Ballistics and with results from other simulations.

The goal was to evaluate the accuracy of simulation results and generate a comprehensive drag coefficient graph for specific rocket designs.
