Computer Science Interests

High-Performance Computing & Optimization

I'm deeply interested in high-performance computing, particularly C++ optimizations that push hardware to its limits. My interests include:

  • Multi-threaded application development
  • Performance optimization techniques
  • Low-level programming and memory management
  • C and Assembler level optimizations

Through my physics simulator project, I've explored techniques for maximizing computational efficiency while maintaining numerical accuracy.

Embedded Systems & Real-time Computing

I'm fascinated by embedded systems programming, especially in mission-critical applications like rocket control systems. My focus areas include:

  • Real-time operating systems (FreeRTOS)
  • Sensor data integration and processing
  • Firmware development with ESP-IDF
  • Resource-constrained programming

I enjoy the challenge of creating reliable software that must operate within strict timing and resource constraints.
